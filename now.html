side experiments
A steady cadence of in-house builds — mostly MCP tooling and coding-agent tuning. The goal is craft: get faster at the parts of the loop that are ours to control.
  • Custom MCP servers — wiring local context, internal docs, and project state into the agents the team uses daily.
  • Coding-agent optimization — prompt scaffolds, subagent orchestration, memory and planning patterns that hold up across long sessions.
  • Eval harnesses for our own workflows — measuring agent reliability on real tasks instead of trusting vibes.
  • Full-stack scratch builds (React, Node, Flask) and a few distributed-systems holdovers — smaller scope, useful for thinking.
